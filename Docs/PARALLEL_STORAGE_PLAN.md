## Performance Targets

| Metric | Current | Target (Parallel) | Improvement |
|--------|---------|-------------------|-------------|
| Insert 10K rows (1 thread) | ~130ms | ~130ms | Same |
| Insert 10K rows (4 threads) | ~130ms* | ~50ms | **2.6x** |
| Insert 10K rows (8 threads) | ~130ms* | ~35ms | **3.7x** |
| Mixed read/write (4 threads) | ~200ms | ~80ms | **2.5x** |

*Currently parallel = sequential due to global lock
