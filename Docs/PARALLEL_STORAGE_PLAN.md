## API Usage

### WitDatabaseBuilder API

```csharp
// Simple auto mode
var db = new WitDatabaseBuilder()
    .WithFilePath("db.witdb")
    .WithBTree()
    .WithParallelWrites()
    .Build();

// With specific mode and options
var db = new WitDatabaseBuilder()
    .WithFilePath("db.witdb")
    .WithLsmTree()
    .WithParallelWrites(ParallelMode.Buffered)
    .WithMaxWriters(8)
    .Build();

// With full options
var db = new WitDatabaseBuilder()
    .WithFilePath("db.witdb")
    .WithBTree()
    .WithParallelWrites(opts => {
        opts.Mode = ParallelMode.Latched;
        opts.MaxWriters = 8;
        opts.UseOptimisticReads = true;
        opts.TrackStatistics = true;
    })
    .Build();
```

### ADO.NET API

```csharp
// Connection string
var cs = "Data Source=db.witdb;Store=btree;Parallel Mode=Auto;Max Writers=8";
using var conn = new WitDbConnection(cs);
conn.Open();

// ConnectionStringBuilder
var csBuilder = new WitDbConnectionStringBuilder {
    DataSource = "db.witdb",
    ParallelMode = WitDbParallelMode.Latched,
    MaxWriters = 4
};
using var conn = new WitDbConnection(csBuilder.ConnectionString);
```

### EF Core API

```csharp
// Via connection string
services.AddDbContext<MyContext>(options =>
    options.UseWitDb("Data Source=db.witdb;Parallel Mode=Auto"));

// Via builder
services.AddDbContext<MyContext>(options =>
    options.UseWitDb("Data Source=db.witdb", witOpts => {
        witOpts.UseParallelWrites(WitDbParallelMode.Latched);
        witOpts.MaxWriters(8);
    }));
```

### KeyValueStoreFactory API

```csharp
// Direct store creation
var options = new ParallelModeOptions {
    Mode = ParallelMode.Auto,
    MaxWriters = 8,
    TrackStatistics = true
};

using var btreeStore = KeyValueStoreFactory.CreateBTreeStore("db.witdb", options);
using var lsmStore = KeyValueStoreFactory.CreateLsmStore("./lsm_data", options);
```
